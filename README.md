### Project Description:
Invoice Generator is a billing application developed in Java using the Swing framework. It allows users to fill in invoice details, add multiple items, apply optional tax and discount, and generate a formatted invoice summary.

### Features
- Enter company name, customer name, invoice number, and date
- Select payment method (Cash, Credit Card, Bank Transfer, Cheque)
- Mark invoice as Paid or Unpaid
- Add and remove line items with quantity and unit price
- Optional 10% Tax and 5% Discount with live summary update
- Generate a formatted invoice preview
- Full input validation with custom exceptions
- JUnit 4 unit tests for business logic

##  Setup Instructions

### Run in Eclipse

1. Clone the repository:
   ```
   git clone https://github.com/hibajavaid/InvoiceGenerator.git
   ```
2. Open Eclipse → `File` → `Import` → `General` → `Existing Projects into Workspace`
3. Browse to the `InvoiceGenerator` folder → Click **Finish**
4. Add JUnit 4 if needed: Right-click project → `Build Path` → `Add Libraries` → `JUnit 4`
5. Run: Right-click `GUI.java` → `Run As` → `Java Application`
6. Test: Right-click `InvoiceCalculatorTest.java` → `Run As` → `JUnit Test`
