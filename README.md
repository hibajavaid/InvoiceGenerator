Invoice Generator is a simple yet functional billing application developed in Java. It provides a clean graphical interface for adding items to an invoice, applying tax and discount options, and calculating the grand total in real time.

Project Structure:

├── src/
│   └── InvoiceGenerator/
│       ├── GUI.java                  # Main GUI + Event Handling
│       └── InvoiceCalculatorTest.java# JUnit Test Cases
├── .settings/

│   └── org.eclipse.jdt.core.prefs
